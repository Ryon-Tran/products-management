# Product Management Project

Quản lý sản phẩm, giỏ hàng, thanh toán, địa chỉ giao hàng, quản trị viên, và giao diện hiện đại lấy cảm hứng từ Shopee.

## Cấu trúc thư mục
- `backend/`: API Express, PostgreSQL, quản lý sản phẩm, giỏ hàng, đơn hàng, người dùng, địa chỉ.
- `frontend/`: Next.js, giao diện người dùng, quản lý sản phẩm, giỏ hàng, thanh toán, đăng nhập, đăng ký, quản trị viên.

## Hướng dẫn sử dụng
1. Cài đặt dependencies:
   - Backend: `cd backend && npm install`
   - Frontend: `cd frontend && npm install`
2. Khởi động backend: `npm run dev` (mặc định port 4000)
3. Khởi động frontend: `npm run dev` (mặc định port 3000)
4. Truy cập giao diện tại `http://localhost:3000`

## Tính năng nổi bật
- Quản lý sản phẩm, giỏ hàng, thanh toán, địa chỉ giao hàng
- Giao diện hiện đại, tối giản, responsive
- Tìm kiếm, lọc, phân loại sản phẩm
- Đăng nhập, đăng ký, quản trị viên
- Thông báo, xác thực, bảo mật

## Công nghệ sử dụng
- Backend: Node.js, Express, PostgreSQL
- Frontend: Next.js, React, ShadCN UI
